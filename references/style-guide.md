# 小红书内容创作风格指南

## 内容风格特点

### 整体调性

- **真诚分享**：像朋友间的对话，不是硬广告
- **有价值**：提供实用信息或情绪价值
- **有共鸣**：让读者产生"这说的就是我"的感觉
- **易读性**：短句、分段、符号辅助

### 语言风格

**推荐**：
- 口语化表达
- 第一人称叙述
- 感叹句增加情绪
- 疑问句引发互动

**避免**：
- 官方腔调
- 过于书面化
- 长句堆砌
- 生硬推销

## 标题写作技巧

### 黄金公式

```
[数字/人称] + [痛点/好奇点] + [解决方案/结果]
```

### 高点击率标题类型

| 类型 | 示例 |
|------|------|
| 数字型 | 「5个让我后悔没早知道的效率工具」 |
| 疑问型 | 「为什么大家都在用这个AI工具？」 |
| 对比型 | 「用了这个方法，效率提升3倍」 |
| 否定型 | 「别再这样用ChatGPT了」 |
| 时效型 | 「2024最新！这个功能太好用了」 |
| 共情型 | 「终于找到了！一直想要的XXX」 |

### Emoji使用规范

**推荐位置**：
- 标题开头或结尾
- 重点词前后

**推荐数量**：1-2个

**常用Emoji**：
- 惊喜/发现：✨ 💡 ⭐
- 推荐/喜爱：❤️ 💖 👍
- 警示/注意：⚠️ 📢 ‼️
- 分享/学习：📚 💻 🎯

## 正文结构模板

### 字数要求

**必须在 300-500 字之间**，确保内容充实有料。

字数分配建议：
- 开头钩子：30-50字
- 核心内容：200-350字
- 总结段落：30-50字
- 互动引导：20-30字

## 排版规范

### 段落设置

- 每段2-4行
- 段落间空一行
- 避免大段文字堆砌

### 分隔符使用

```
—————————————
· · · · · ·
▪️▪️▪️▪️▪️
━━━━━━━━━━
```

### 列表格式

```
✅ 推荐做法
❌ 不推荐做法

1️⃣ 2️⃣ 3️⃣ （步骤序号）
• （普通列表项）
```

## 标签策略

### 标签组成（建议8-10个）

1. **核心标签**（2-3个）：与主题直接相关
2. **热门标签**（2-3个）：当前热度高的话题
3. **长尾标签**（2-3个）：精准但竞争小
4. **场景标签**（1-2个）：使用场景相关

### 标签格式

```
#关键词 #关键词 #关键词
```

### AI相关热门标签参考


## 配图设计原则

### 数量要求

**必须 3-6 张配图**：
- 第1张：封面图（必须）
- 第2-4张：内容图（详细展示要点）
- 第5张：总结图（可选）
- 第6张：互动图（可选）

### 尺寸比例

- **必须**：3:4 竖版（小红书主流）
- 推荐尺寸：1080×1440px 或 360×480px

### 减少AI感的设计技巧

#### 1. 场景真实化
- 使用办公桌、书架、咖啡厅等真实场景背景
- 加入生活化元素：咖啡杯、便签纸、手机、键盘
- 避免纯色渐变背景

#### 2. 色调自然
- 使用莫兰迪色系（低饱和度）
- 自然光色调，避免过于鲜艳
- 推荐配色：
  - 奶油白 + 浅灰 + 点缀色
  - 米色 + 棕色 + 绿色
  - 浅蓝 + 白色 + 深蓝

#### 3. 排版随意
- 不要过于完美对称的构图
- 元素可以有轻微的倾斜或错位
- 保留"随手拍"的感觉

#### 4. 字体选择
- 标题：思源黑体、苹方粗体
- 正文：思源宋体、苹方常规
- 避免过于艺术化的字体

### 设计要素

1. **主标题**：大字、醒目、3-8个字
2. **副标题**（可选）：补充说明
3. **视觉焦点**：相关图像或图标
4. **品牌调性**：统一风格

### 配色建议

| 主题类型 | 推荐配色 |
|---------|---------|
| 科技/效率 | 蓝色系、紫色系 |
| 生活/分享 | 暖色系、粉色系 |
| 教程/知识 | 白色系、简约风 |
| 种草/推荐 | 橙色系、黄色系 |

### 字体建议

- 标题：粗体、无衬线
- 正文：常规、易读
- 避免超过2种字体

## 内容禁忌

### 平台规则

- 不发虚假信息
- 不发违规内容
- 不过度营销
- 不抄袭搬运

### 内容质量

- 不做标题党（内容要匹配）
- 不灌水（要有实际价值）
- 不重复发布相同内容

## 互动技巧

### 引导互动的句式

```
"你们觉得呢？"
"评论区告诉我~"
"有同款的姐妹吗？"
"你最喜欢哪个？"
"还想看什么内容？"
"收藏起来慢慢看！"
"记得点赞收藏，不然就找不到啦！"
"有问题可以评论区问我~"
```

### 回复评论建议

- 及时回复（提升互动率）
- 真诚交流（不敷衍）
- 引导话题（延续讨论）
